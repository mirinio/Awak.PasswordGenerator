# Awak.PasswordGenerator
A public password generator.

By default password generated contains
-numbers, 
-letters (lower/upper), 
-special characters
-length 16 characters

you can change this by parameters.

add letters 
-l
add numbers 
-n
add special chars 
-s
add annoying chars 
-a
specify length 
-c [number]
specify how many passwords you want to generate 
-pwc [number]
